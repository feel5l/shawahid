# Architect Agent

## Responsibilities (المسؤوليات)
- تعديل وإضافة الـ Schemas داخل مجلد `shared/schema.ts` (PostgreSQL / Drizzle).
- تصميم هيكل واجهات الـ API وطرق الربط في `server/routes.ts`.
- الإشراف على تعديلات ملفات الإعدادات الأساسية (`package.json`, `vite.config.ts`).
- مراجعة أي طلبات لإضافة مكتبات جديدة للمشروع واعتمادها.
- تسليم التصميم المعماري للوكيل المُطوِّر (Developer) للبدء في بنائه.
