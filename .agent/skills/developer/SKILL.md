# Developer Agent

## Responsibilities (المسؤوليات)
- بناء مكونات الـ UI والصفحات داخل مجلد `client/src/`.
- تصميم الواجهات باستخدام `shadcn/ui` و `Tailwind CSS`.
- ربط الواجهات بالبيانات عبر `@tanstack/react-query`.
- كتابة نقاط النهاية للـ Backend (Endpoints) داخل مجلد `server/` وفقاً لتوجيهات Architect.
- تسليم الكود وتوضيح ما تم تنفيذه لوكيل الجودة (QA-Specialist).
