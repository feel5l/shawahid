- **Input:** Receive `mobileNumber`, `fullNameArabic`, etc.
- **Logic:**
  1. Check if a user with this `mobileNumber` ALREADY exists (let's call it `oldUser`).
  2. If `oldUser` exists AND it is different from `currentUser`:
     - Update `oldUser` with the new profile data (name, school, etc.).
     - **Session Migration:** Manually update `req.session.passport.user` to `oldUser.id`.

     - Save session: `await new Promise((resolve) => req.session.save(resolve));`
     - Return the `oldUser` data.
  3. If no `oldUser` exists:
     - Update `currentUser` with the `mobileNumber` and profile data normally.
