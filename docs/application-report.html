نظام توثيق شواهد الأداء الوظيفي

📅 تاريخ التقرير: أبريل 2026 👨‍💻 المطور: عبدالعزيز الخلفان 🏷 الإصدار: 1.0.0 📂 المستودع: feel5l/shawahid
١ الملخص التنفيذي
نظام توثيق شواهد الأداء الوظيفي هو منصة ويب متكاملة مصممة خصيصًا للبيئة التعليمية السعودية، تتيح للمعلمين توثيق مؤشراتهم المهنية وشواهدهم الوظيفية بصورة رقمية منظمة، مع توفير دورة اعتماد واضحة تبدأ من المعلم وتنتهي بمراجعة المدير واعتماده. النظام مبني بتقنيات حديثة ويعمل بشكل مستقر في بيئة الإنتاج.
٢ الفئات المستفيدة
الفئة الصلاحية الحالة
🧑‍🏫 المعلمون والمعلمات توثيق المؤشرات وإرسالها للاعتماد مفعّل
🏫 مدير/ة المدرسة مراجعة المؤشرات واعتمادها أو ردّها مفعّل
🔍 المشرفون التربويون متابعة الأداء على مستوى أشمل قيد التطوير
⚙️ منشئ النظام إدارة كاملة للنظام والمستخدمين مفعّل
٣ الوظائف الأساسية

🧑‍🏫 واجهة المعلم

  • إنشاء وإدارة مؤشرات الأداء
  • تحديد وإكمال معايير الأداء
  • رفع الشواهد والمستندات
  • ربط الأداء بالاستراتيجيات التعليمية
  • إرسال المؤشرات للاعتماد
  • متابعة حالة كل مؤشر

🏫 واجهة المدير

  • لوحة تحكم بإحصائيات المدرسة
  • عرض قائمة جميع المعلمين
  • مراجعة المؤشرات المرفوعة
  • اعتماد أو رفض المؤشرات بملاحظات
  • متابعة التقدم العام للمدرسة
  • تقارير الأداء المجمّعة

٥ البنية التقنية
🖥 الواجهة الأمامية React 18 + TypeScript + Vite
⚙️ الخادم (Backend) Express.js + Node.js + TypeScript
🗄 قاعدة البيانات PostgreSQL (Neon Serverless)
🔐 المصادقة Session-based + Passport.js + OAuth
🎨 واجهة UI Shadcn UI + Tailwind CSS (RTL)
☁️ النشر Netlify / Render.com
٦ الجوانب الأمنية
  • جلسات آمنة مشفرة مخزنة في قاعدة البيانات
  • التحقق الكامل من صحة جميع المدخلات عبر مكتبة Zod
  • عزل بيانات كل مستخدم — لا يرى المعلم إلا بياناته الخاصة
  • نظام أدوار متعدد المستويات يمنع الوصول غير المصرح به
  • دعم اتصالات SSL/TLS في بيئة النشر
  • الأسرار والمفاتيح مُدارة عبر متغيرات البيئة فقط
٧ القيمة الإدارية المضافة
تقليل الأعمال الورقية اليدوية في توثيق الأداء
رفع جودة الحوكمة والشفافية في المدرسة
تسريع دورة المراجعة والاعتماد بين المعلم والمدير
أرشفة رقمية دقيقة لجميع المؤشرات والشواهد
لوحة إحصائيات مباشرة لدعم اتخاذ القرار
قابلية التوسع لاستيعاب المزيد من المدارس والمستخدمين
٨ التوصيات والخطوات المقترحة
١
اعتماد منصة نشر موحدة: تحديد Netlify أو Render رسميًا لتوحيد إجراءات الصيانة والتحديث.
٢
سياسة نسخ احتياطي دورية: جدولة نسخ احتياطية تلقائية لقاعدة بيانات PostgreSQL أسبوعيًا.
٣
تفعيل دور المشرف التربوي: استكمال تطوير صلاحيات المشرف لاستيعاب الإشراف على مستوى أشمل.
٤
تقارير دورية موسعة: إضافة تقارير تتضمن نسبة إكمال المعلمين وزمن دورة الاعتماد.
٩ الخلاصة
النظام جاهز تشغيليًا ويغطي دورة توثيق الأداء المهني من الإنشاء حتى الاعتماد بصورة متكاملة. بنيته التقنية حديثة وقابلة للتوسع، وتصميمه مُحكم يلائم البيئة التعليمية السعودية. يُوصى بالمضي قدمًا في تشغيله رسميًا والبدء بتطبيق التوصيات المذكورة أعلاه لضمان الاستمرارية والتطوير.
